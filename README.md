# Groq Chat

This small project uses the Groq API. To provide your API key locally, copy `.env.example` to `.env` and set your key:

```
GROQ_API_KEY=gsk_your_key_here
```

The app will load the key from the environment automatically. You can also enter a key manually in the sidebar UI.

Run locally with:

```powershell
C:/Users/arjun/AppData/Local/Python/pythoncore-3.14-64/python.exe -m pip install -r requirements.txt
C:/Users/arjun/AppData/Local/Python/pythoncore-3.14-64/python.exe -m streamlit run chatbot.py
```
